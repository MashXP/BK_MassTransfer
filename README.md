# BK_MassTransfer

This repository contains materials related to the study of Mass Transfer.

## Project Structure:
- `hcmut.sty`: LaTeX style file for the project.
- `mass_transfer_main.tex`: The main LaTeX document for the mass transfer course.
- `Sections/`: Contains individual LaTeX source files for different sections of the main document.
- `Images/`: Stores images used throughout the LaTeX documents.
- ~~`Materials of Mass Transfer/`: Contains various study materials, primarily in PDF format, which have undergone or can undergo OCR processing.~~ 
- `quiz/`: Contains files for a my web-based quiz application: https://mashxp.github.io/quiz_webapp/
